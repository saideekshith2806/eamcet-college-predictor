📁 How to plug in your real dataset

Open this HTML file, scroll to the <script> section and find const DATA = [...]. Each entry looks like:

{ college: "Full College Name", short: "Short, Location", branch: "CSE",
  district: "Hyderabad", type: "Government",
  cutoffs: {
    OC:  [2016_rank, 2017_rank, ..., 2024_rank],
    SC:  [2016_rank, ..., 2024_rank],
    BC_A:[2016_rank, ..., 2024_rank]
  }}

Replace the sample rows with your 2016–2024 CSV. For 2025 data (not yet released): use the average of 2022–2024 as an estimate and note it in the UI disclaimer.
